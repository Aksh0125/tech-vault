# Airline Management System

A Full Stack Airline Management Application built using **C#**, **.NET**, **Java**, **MySQL**

## Tech Stack 
-> Backend 1-- C# .NET
-> Backend 2 -- Java
-> Datbase -- MySQL
-> Frontend -- .NET Web App

## Project Structure
-> backend-dotnet -- .NET Web API
-> backend-Java -- Java Backend Services
-> Database -- MySQL schema & Scripts

## How to RUN
1. Import database scripts from  `/databse`
2. Run .NET backend using Visual Studio
3. Run Java Backend using Eclipse / IntelliJ
4. Update database credentials in config files

## Features
-> User Authentication
-> Role Based Access
-> Management Operations
-> Secure Databse Connectivity
